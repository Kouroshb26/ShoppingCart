# ShoppingCart
Shopping cart app that keeps track of your items when shopping. 

##Functionality:

  -Can Create Lists<br>
  -Icons appear on most widly used items<br>
  -Swipe right to Highlight the item when item has been bought<br>
  -Swipe left to delete items<br>
